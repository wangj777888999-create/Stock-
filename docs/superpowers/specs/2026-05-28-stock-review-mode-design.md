# 个股快速复盘模块 — 设计文档

**日期：** 2026-05-28  
**状态：** 待审阅

---

## 背景与目标

用户需要通过大量复盘来锻炼盘感，要求能快速逐一浏览全市场个股图表（日/周/月 K 线、分时线、均线、量能），并在浏览过程中一键加入自选股或记录备注。现有技术分析页针对单只股票查询，缺乏批量翻阅能力。

---

## 范围

- **支持市场：** A 股（stock_list.json 内的全部股票）
- **不涉及：** 港股、美股、期货、加密货币
- **不修改：** 现有技术分析页、自选股模块

---

## 整体架构

新增独立侧边栏入口"复盘"，对应 SPA 视图 `view-review`，与其他页面平级。

### 视图布局

```
┌──────────────────────────────────────────────────────┐
│  ← 上一只   贵州茅台 (600519)   342 / 5000   下一只 → │  顶部导航栏
├──────────────────────────────────────────────────────┤
│  [分时] [日线] [周线] [月线]          [⛶ 放大]       │  周期 Tab
│                                                      │
│                   主图（K线 + 均线）                   │  60% 高度
│                                                      │
├──────────────────────────────────────────────────────┤
│                   量能（成交量柱）                     │  20% 高度
├──────────────────────────────────────────────────────┤
│  ★ 加入自选股      ✏ 写备注: [_______________] [保存]  │  操作栏
│  已有备注列表（按时间倒序，每条可删除）                │
└──────────────────────────────────────────────────────┘
```

### 股票队列策略

- 进入复盘视图时，从 `window.STOCK_LIST`（stock_list.json 内嵌数据）读取全部 A 股
- 前端执行 Fisher-Yates 随机洗牌，生成本次复盘顺序 `reviewQueue[]`
- 队列存于 JS 内存，刷新页面后重新洗牌
- 默认从 index 0 开始，支持前进/后退

## 后端变更

### 新增数据库表

在 `database.py` 的 `_migrate()` 中追加：

```sql
CREATE TABLE IF NOT EXISTS review_notes (
    id         INTEGER PRIMARY KEY AUTOINCREMENT,
    symbol     TEXT NOT NULL,
    name       TEXT,
    note       TEXT NOT NULL,
    created_at TEXT NOT NULL DEFAULT (datetime('now'))
);
```

## 前端交互逻辑

### 复盘队列初始化

```
initReview():
  1. 从 window.STOCK_LIST 读取全部股票
  2. Fisher-Yates 洗牌 → reviewQueue[]
  3. reviewIndex = 0
  4. loadReviewStock(reviewQueue[0])
```

### 股票加载流程

```
loadReviewStock(symbol):
  1. 更新顶部导航：股票名 + 代码 + 进度（index+1 / total）
  2. 按当前激活 Tab 请求 K 线或分时数据
  3. 渲染主图（K线 + MA5/MA10/MA20/MA60）+ 量能副图
  4. 请求 GET /api/review/notes?symbol=xxx，展示已有备注
  5. 清空备注输入框
```

### 键盘快捷键

（仅在复盘视图激活时生效）

| 按键 | 操作 |
|------|------|
| `→` / `Space` | 下一只 |
| `←` | 上一只 |
| `1` | 切换到分时 |
| `2` | 切换到日线 |
| `3` | 切换到周线 |
| `4` | 切换到月线 |
| `W` | 加入自选股 |
| `N` | 聚焦备注输入框 |

### 备注交互

- 输入框 + 保存按钮，`Enter` 也可提交
- 同一只股票支持多条备注，按时间倒序展示
- 每条备注右侧有删除按钮
- 保存成功后输入框清空，备注列表刷新

### 放大模式

点击 `⛶` 按钮调用 `toggleSidebar()`（复用现有侧边栏收起逻辑），图表容器自动撑满宽度。

### 图表渲染

复用现有技术分析页的 K 线渲染函数，封装为共享函数供复盘视图调用，避免重复代码。均线默认显示 MA5/MA10/MA20/MA60。

## 验证方案

1. 启动 `python app.py`，`/docs` 页面可见 `/api/review/*` 接口
2. 侧边栏出现"复盘"入口，点击进入视图
3. 默认日线图正确渲染，MA 线和量能柱显示正常
4. 键盘 `→` / `←` 切换股票，进度计数器随之变化
5. Tab 切换：分时 / 日线 / 周线 / 月线 各自正确请求并渲染
6. 点击"加入自选股"，在自选股页面验证已加入
7. 写入备注，刷新后再次进入同一只股票，备注仍然显示
8. 删除备注，列表立即更新
9. 点击放大按钮，侧边栏收起，图表区宽度增大
