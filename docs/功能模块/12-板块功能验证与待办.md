## 二、前端冒烟测试（待浏览器验证）

在浏览器中打开 http://localhost:8000，依次验证以下交互路径：

- [ ] 侧边栏出现"板块"按钮，点击后切换到板块视图
- [ ] 行业板块 Tab 显示热力方块图，方块按涨跌幅红绿着色，面积随成交额变化
- [ ] 点击"概念板块"Tab，数据切换为概念板块列表
- [ ] 点击"全部"Tab，行业 + 概念板块合并显示
- [ ] 点击热力图中的方块 → 进入板块详情，显示 K 线图 + 成分股列表
- [ ] 日 K / 周 K / 月 K 切换正常
- [ ] 点击成分股中的股票行 → 跳转到股票查询视图并加载该股票详情
- [ ] 点击"← 返回板块列表" → 热力图和列表重新显示

## 三、已知限制

| 限制 | 说明 | 后续方案 |
|------|------|----------|
| East Money API 不可用 | `17.push2.eastmoney.com` 在当前网络被拒连 | 如有 VPN/代理可解除则自动恢复 |
| 成分股无降级源 | akshare 无同花顺板块成分股接口 | 待 akshare 更新或接入其他数据源 |
| 概念板块仅 name/code | THS 概念无汇总行情接口 | 可考虑用 iwencai 自然语言补充 |
| K 线无涨跌幅列 | THS K 线不提供涨跌幅，从收盘价计算 | 首条数据无涨跌幅（正常） |

## 四、后续优化建议

1. **成分股降级**：探索 `stock_board_concept_info_ths` 是否可获取概念板块成分股
2. **概念板块行情**：考虑用 East Money 概念板块 spot 接口补充行情数据（当网络可用时）
3. **前端模块化**：板块视图代码待前端重构时迁移为独立组件
4. **排序交互**：前端添加表头点击排序（当前仅后端排序 + 默认按涨跌幅降序）
