## 一、前端内存泄漏修复

### 1.1 问题根因

`index.html` 中共 6 处 `ResizeObserver` 在创建时未保存引用，导致无法在视图切换或图表重建时调用 `disconnect()`，产生 Detached DOM 节点持续累积。

| # | 函数 | 问题 |
|---|------|------|
| L1 | `loadKline()` — MACD 子图 | `new ResizeObserver()` 匿名创建，无法释放 |
| L2 | `loadKline()` — RSI 子图 | 同上 |
| L3 | `selectFund()` | 每次切换基金重建 observer，旧的泄漏 |
| L4 | `switchFundKlinePeriod()` | 每次切换周期重建 observer，旧的泄漏 |
| L5 | `selectFutures()` | 每次切换期货重建 observer，旧的泄漏 |
| L6 | `switchFuturesKlinePeriod()` | 每次切换周期重建 observer，旧的泄漏 |

> 行号为当前版本近似参考，实现时以实际搜索为准。

### 1.2 修复方案

**步骤 1 — 声明全局变量**（在全局变量区统一新增）

```javascript
let _klineMacdObserver = null;
let _klineRsiObserver = null;
let _fundResizeObserver = null;
let _futuresResizeObserver = null;
```

**步骤 2 — 修改 `_disposeKlineSubCharts()`**

在销毁 K 线子图时，先 disconnect observer，再移除图表实例：

```javascript
function _disposeKlineSubCharts() {
  if (_klineMacdObserver) { _klineMacdObserver.disconnect(); _klineMacdObserver = null; }
  if (_klineRsiObserver)  { _klineRsiObserver.disconnect();  _klineRsiObserver = null;  }
  if (_klineMacdChart) { try { _klineMacdChart.remove(); } catch(e){} _klineMacdChart = null; }
  if (_klineRsiChart)  { try { _klineRsiChart.remove();  } catch(e){} _klineRsiChart = null;  }
  // 原有 DOM 清理继续...
}
```

**步骤 3 — 修改 `loadKline()` 中的 MACD/RSI observer**

```javascript
// MACD（约 L3526）
_klineMacdObserver = new ResizeObserver(() => {
  if (_klineMacdChart) _klineMacdChart.applyOptions({ width: macdEl.clientWidth });
});
_klineMacdObserver.observe(macdEl);

// RSI（约 L3561）
_klineRsiObserver = new ResizeObserver(() => {
  if (_klineRsiChart) _klineRsiChart.applyOptions({ width: rsiEl.clientWidth });
});
_klineRsiObserver.observe(rsiEl);
```

**步骤 4 — 修改基金图表函数**（`selectFund()` 和 `switchFundKlinePeriod()`）

两个函数顶部均先 disconnect 旧 observer，再创建新图表后保存新引用：

```javascript
// 函数顶部
if (_fundResizeObserver) { _fundResizeObserver.disconnect(); _fundResizeObserver = null; }

// 创建新图表后（替换原匿名 new ResizeObserver）
_fundResizeObserver = new ResizeObserver(() => {
  if (_fundChart) _fundChart.applyOptions({ width: chartEl.clientWidth });
});
_fundResizeObserver.observe(chartEl);
```

**步骤 5 — 修改期货图表函数**（`selectFutures()` 和 `switchFuturesKlinePeriod()`）

同步骤 4，使用 `_futuresResizeObserver`：

```javascript
// 函数顶部
if (_futuresResizeObserver) { _futuresResizeObserver.disconnect(); _futuresResizeObserver = null; }

// 创建新图表后
_futuresResizeObserver = new ResizeObserver(() => {
  if (_futuresChart) _futuresChart.applyOptions({ width: chartEl.clientWidth });
});
_futuresResizeObserver.observe(chartEl);
```

**步骤 6 — 在 `switchView()` 顶部统一清理**

视图切换时作为最后防线，确保跨页面的残留 observer 全部释放：

```javascript
function switchView(view, el) {
  if (_klineMacdObserver)   { _klineMacdObserver.disconnect();   _klineMacdObserver = null;   }
  if (_klineRsiObserver)    { _klineRsiObserver.disconnect();    _klineRsiObserver = null;    }
  if (_fundResizeObserver)  { _fundResizeObserver.disconnect();  _fundResizeObserver = null;  }
  if (_futuresResizeObserver){ _futuresResizeObserver.disconnect(); _futuresResizeObserver = null; }

  // 附加：离开自选股视图时清理定时器
  if (_watchlistInterval && view !== 'watchlist') {
    clearInterval(_watchlistInterval);
    _watchlistInterval = null;
  }

  // 原有逻辑继续...
}
```

### 1.3 实现注意事项

- disconnect 必须在创建新 observer **之前**执行，顺序不可颠倒。
- 步骤 2 和步骤 6 形成双重保障：图表层在销毁时清理，视图层在切换时兜底。
- 不引入新的抽象（如 observer 注册表），保持与现有单文件风格一致。

### 1.4 验收标准

DevTools → Memory 面板 → 反复切换视图 10 次（股票 → 基金 → 期货 → 股票…），Detached 节点数不持续增长。

## 二、北向资金 + 龙虎榜

### 2.1 后端实现

追加到 `src/routers/market.py`（现有约 71 行），复用现有 `cache_get` / `cache_set` 模式。

#### 路由表

| 路由 | AKShare 函数 | 缓存 TTL | 说明 |
|------|-------------|---------|------|
| `GET /api/market/north-flow` | `ak.stock_hsgt_fund_min_em(symbol="北向资金")` | 300s | 今日分钟级净流入，取最近 50 条 |
| `GET /api/market/north-history` | `ak.stock_hsgt_hist_em(symbol="北向资金")` | 300s | 历史日线净买额 |
| `GET /api/market/dragon-tiger` | `ak.stock_lhb_detail_daily_sina(date=date)` | 600s | 当日龙虎榜，支持 `?date=YYYYMMDD` |

#### 路由实现规范

```python
from stock_utils import cache_get, cache_set

@router.get("/api/market/north-flow")
async def get_north_flow():
    cached = cache_get("north-flow")
    if cached is not None:
        return {"success": True, "data": cached}
    try:
        import akshare as ak
        df = ak.stock_hsgt_fund_min_em(symbol="北向资金")
        data = df.tail(50).to_dict(orient="records")
        cache_set("north-flow", data, 300)
        return {"success": True, "data": data}
    except Exception as e:
        return {"success": False, "error": str(e)}


@router.get("/api/market/north-history")
async def get_north_history():
    cached = cache_get("north-history")
    if cached is not None:
        return {"success": True, "data": cached}
    try:
        import akshare as ak
        df = ak.stock_hsgt_hist_em(symbol="北向资金")
        data = df.tail(30).to_dict(orient="records")
        cache_set("north-history", data, 300)
        return {"success": True, "data": data}
    except Exception as e:
        return {"success": False, "error": str(e)}


@router.get("/api/market/dragon-tiger")
async def get_dragon_tiger(date: str = ""):
    cache_key = f"dragon-tiger-{date}" if date else "dragon-tiger"
    cached = cache_get(cache_key)
    if cached is not None:
        return {"success": True, "data": cached, "date": date}
    try:
        import akshare as ak
        kwargs = {"date": date} if date else {}
        df = ak.stock_lhb_detail_daily_sina(**kwargs)
        data = df.to_dict(orient="records") if df is not None and not df.empty else []
        cache_set(cache_key, data, 600)
        return {"success": True, "data": data, "date": date}
    except Exception as e:
        return {"success": False, "error": str(e), "data": []}
```

> 龙虎榜非交易日 AKShare 返回空 DataFrame，后端统一返回 `data: []`，前端负责展示空状态。  
> `north-flow` 在非交易时段 AKShare 可能返回空，前端同样用空状态处理，不报错。
