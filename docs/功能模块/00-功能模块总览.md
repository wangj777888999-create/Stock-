## 已解决的关键技术问题

### K 线数据（06-K线数据断路根因分析）
五大故障模式全部修复：
- F1：数据源切换（AKShare 主，腾讯备）
- F2：日期范围不足（daily=730天，weekly/monthly=3650天）
- F3：数据有效性校验（过滤全 None OHLC 记录）
- F4：TTL 分级（K线从 5min 延长至 1h）
- F5：API 参数修正

### 板块行情（13-板块页面优化设计）
- 热力图：CSS Grid 均匀网格，7色阶，替代原来大小不一的 flex 布局
- 成分股：补充 PE动、流通市值、换手率字段
- K线：日线2年历史，三源降级（东方财富→同花顺→pywencai）

### 代码质量（04-代码质量审查报告）
已修复的关键问题：
- ✅ WebSocket URL 白名单校验（防 SSRF）
- ✅ Cookie 明文存储风险（环境变量隔离）
- ✅ asyncio.gather 异常吞并（return_exceptions 处理）
- ✅ Playwright 资源泄漏（browser.close）

仍待处理：
- 🔲 stock_service.py 1011行，需拆分
- 🔲 单元测试覆盖
- 🔲 ORM 或参数化 SQL 统一
