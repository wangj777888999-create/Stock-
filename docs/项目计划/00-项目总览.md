## 项目是什么

**StockPulse — 个人投资信息平台**

核心链路：监控微信公众号博主 → 抓取文章 → AI 多视角分析 → 跟踪推荐股票 → 模拟/实盘交易 → 胜率统计

差异化：三视角 AI 分析（价值派/成长派/趋势派）+ 交易感知闭环 + MCP 协议

## 下一步优先级

### 🔴 近期（值得马上做）

1. **博主评分排行榜（V2 M2-M5）**
   - 推荐准确率、盈亏比统计已有数据基础（`recommendation_scores` 表）
   - 是本项目最核心的差异化功能延伸
   - 参考：`10-StockPulse-V2设计文档.md`

2. **前端内存泄漏修复（Phase 2）**
   - 6处内存泄漏（WebSocket 未关闭、setInterval 未清除）
   - 长时间使用会导致页面变慢
   - 参考：`09-前端重构改进方案.md`

### 🟡 中期（有时间再做）

3. **北向资金 / 龙虎榜**
   - AKShare 有现成接口，开发成本低
   - A股用户的核心需求

4. **API 认证**
   - 本地使用风险低，但上线前必须做
   - 参考：`03-技术计划TECH_PLAN.md` Phase 6

5. **数据导出（CSV）**
   - 用户高频需求，前端加下载按钮即可

### 🟢 长期（V2 方向）

6. AI 分析升级（YAML 策略 DSL 配置化）
7. 企微/Telegram 通知推送
8. 策略回测引擎
9. 前端 React 迁移
