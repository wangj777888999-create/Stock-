涨跌热力图 — 3 种设计方案

选择你喜欢的风格（点击卡片选中）

A — Treemap（树状热力图）

方块大小按成交额比例，颜色深浅按涨跌幅。主力板块视觉占比最大，符合股票软件常见风格（同花顺 / 东方财富）。

优点

  • 信息密度高
  • 主力板块突出
  • 专业感强

缺点

  • 小板块名称可能被截断

B — 均匀网格（等大方格）

所有板块格子相同大小，按涨跌幅从高到低排列。颜色深浅表示涨跌幅，布局整齐易扫描。

优点

  • 排列整齐
  • 所有名称完整显示
  • 一眼看涨跌排序

缺点

  • 看不出成交额大小

C — 行情面板（条形列表）

类似行情软件的排行榜风格，每行显示板块名、涨跌幅、进度条和成交额。信息最完整、可读性最强。

优点

  • 信息最完整
  • 可读性强
  • 数据对齐清晰

缺点

  • 需要滚动才能看全
  • 热力感弱
