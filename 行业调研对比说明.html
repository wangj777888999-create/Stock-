行业调研 — 四种分析目的对比

底层共用同一套 肖璟六步框架，区别在于 AI 侧重输出的维度不同。定义在 industry_service.py，非 skill。

投资选股

investment

从投资者视角判断"该不该买"。侧重生命周期、竞争格局、估值逻辑、景气度指标。

创业选赛道

startup

从创业者视角判断"能不能做"。侧重商业模式可行性、市场规模（TAM/SAM/SOM）、护城河。

择业找方向

career

从求职者视角判断"值不值得进"。侧重生命周期、行业前景、薪资水平、龙头公司。

完整行研

full

六个维度全覆盖，适合需要全面了解行业的场景。输出最完整、篇幅最长。

底层框架：肖璟六步法

所有四种目的共用同一个 system prompt，AI 按此六步逐步分析
STEP 1
定义行业边界
横向：行业层级（申万/GICS）
纵向：产业链上下游梳理
STEP 2
判断产业生命周期
以渗透率为核心标准
导入期 → 成长期 → 成熟期 → 衰退期
STEP 3
按阶段聚焦分析
3A 可行性（导入期）
3B 规模性（成长期）
3C 防守性（成熟期）
3D 盈利性（成熟期）
STEP 4
估值逻辑
导入期 PS/GMV → 成长期 PEG
成熟期 PE/DCF → 衰退期 PB
STEP 5
PEST 外部因素
政策 / 经济 / 社会 / 技术
当前催化剂或压制因素
STEP 6
景气度跟踪指标
量 / 价 / 利 / 库存 / 预期
推荐数据来源

要点总结

  • 四个选项 不是 skill 定义的，是 industry_service.py 中硬编码的 _PURPOSE_FOCUS 字典。
  • 底层 system prompt 完全相同（肖璟六步框架），区别仅在 user prompt 中附加的"重点输出"指令。
  • 投资选股：最适合炒股场景，聚焦估值 + 景气度。
  • 创业选赛道：最适合评估新业务，聚焦商业可行性 + 市场天花板。
  • 择业找方向：最适合职业规划，聚焦行业前景 + 薪资 + 龙头公司。
  • 完整行研：六步全覆盖，篇幅最长，适合需要全面了解的场景。
